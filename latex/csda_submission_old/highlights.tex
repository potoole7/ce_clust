\section*{Highlights}

\begin{itemize}
    \item Novel framework clustering multivariate tail dependence via conditional extremes.
    \item Uses conditional extremes for flexible, parsimonious tail-dependence models.
    \item Clusters parameters with skew-geometric Jensen--Shannon divergence (closed-form).
    \item Groups similar tail-dependence structures in high-dimensional settings.
    \item Simulations beat existing methods; applied to Irish precipitation \& wind speed data.
\end{itemize}
